（中央社記者楊淑閔台北15日電）中油公司公布，明天16日凌晨零時起，漲各式汽油價格每公升新台幣0.5元、各式柴油價格每公升0.6元。調整後的參考零售價格，中油說，分別為92無鉛汽油每公升24.5元、95無鉛汽油每公升26元、98無鉛汽油每公升28元、超級柴油每公升22.5元。中油表示，除美國境內運轉中原油鑽井機數，創下2011年12月以來的最低數，法國道達爾公司宣布減少支出、出售資產等消息，都讓國際油價繼續上漲。中油並說，本週調價指標7D3B週均價為每桶55.65美元，較前週53.24美元，上漲2.41美元，新台幣兌美元匯率升值新台幣0.015元，國內油價依公式計算漲幅為3.58%，汽、柴油每公升各調漲0.5元、0.6元。1040215
